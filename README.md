## Requirements
- PHP ^7.2.5|^8.0
- Composer
- NodeJs

## Steps
- Clone project "git clone https://github.com/abellimac/a2odev.git"
- Go inside project and run "Composer install"
- Rename the file ".env.example" to ".env"
- Click on the button "Generate app key"
- run "npm run watch"
